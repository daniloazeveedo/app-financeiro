VERSÃO 3.0 RECRIADA — Meu Controle

Arquivos principais:
- index.html
- style.css
- app.js
- manifest.webmanifest
- service-worker.js
- icons/

Para publicar:
1. Extraia o ZIP.
2. Envie os arquivos de dentro da pasta para o GitHub.
3. O index.html precisa ficar na raiz do repositório.
4. Aguarde o GitHub Pages atualizar.
5. No iPhone, remova o ícone antigo e adicione novamente pelo Safari.

Esta versão tem:
- tela inicial no estilo dos prints enviados;
- fluxo de caixa no estilo dos prints enviados;
- PWA;
- exportação/importação Excel;
- backup JSON;
- dados salvos no navegador.
