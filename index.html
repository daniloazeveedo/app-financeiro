Sincronização e Excel

Com Firebase configurado, os dados sincronizam na nuvem. O Excel funciona por exportação/importação.

Como ativar login online

  1. Crie um projeto no Firebase.
  2. Ative Authentication por e-mail/senha.
  3. Ative Firestore Database.
  4. Cole suas chaves em firebase-config.js.
  5. Publique no GitHub Pages, Netlify ou Vercel.
